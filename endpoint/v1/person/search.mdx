## Overview

Search for LinkedIn profiles in our DataLake using advanced filters. This endpoint allows you to find multiple profiles matching specific criteria from our existing database.

<Note>
  **DataLake Search:** This endpoint searches our existing database. Live
  LinkedIn search will be available soon at a different pricing.
</Note>

## Use Cases

<AccordionGroup>
  <Accordion title="Lead Generation">
    Search for potential clients or partners based on job title and company:
    ```json
    {
      "currentPositionTitle": "CTO",
      "companyName": "Tech"
    }
    ```
  </Accordion>

  <Accordion title="Talent Sourcing">
    Find candidates with specific skills and location:
    ```json
    {
      "headline": "React Developer",
      "locationCity": "New York",
      "locationCountryCode": "US"
    }
    ```
  </Accordion>

  <Accordion title="Market Research">
    Identify professionals in specific industries:
    ```json
    {
      "currentPositionTitle": "Product Manager",
      "companyName": "Startup"
    }
    ```
  </Accordion>

  <Accordion title="Network Building">
    Find alumni or people in your network:
    ```json
    {
      "companyName": "Previous Company",
      "locationCity": "Your City"
    }
    ```
  </Accordion>
</AccordionGroup>

## Best Practices

### Efficient Searching

- **Be specific**: Use multiple filters to narrow down results and improve relevance
- **Monitor costs**: Each page costs 0.10 credits (20 results per page). Accessing multiple pages will consume additional credits.
